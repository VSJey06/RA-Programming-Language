* Runtime

* Object System
* Method System

# Major Issues Encountered During v1.0.0

## Issue #001

### REPL Could Not Execute Block Statements

Problem:

Typing:

Db:
...
db.close

inside the REPL executed line-by-line.

Result:

Parser detected:

db.close

outside a Db block.

Error:

Unexpected 'db.close' outside of a Db block

Resolution:

Implemented buffered multi-line REPL execution.

Result:

Entire block is collected before parsing.

Status:

Fixed

---

## Issue #002

### AutoCloseManager Import Failure

Problem:

main.py imported:

from runtime.autoclose import AutoCloseManager

while autoclose.py contained:

class AutoCloser

Error:

ImportError:
cannot import name 'AutoCloseManager'

Resolution:

Updated import references.

Status:

Fixed

---

## Issue #003

### Block Validation Logic Failure

Problem:

Manual block validation inside main.py did not match language rules.

Resolution:

Replaced custom validator with:

AutoCloser().validate(source)

Status:

Fixed

---

## Issue #004

### Class Close Syntax Ambiguity

Original Syntax:

@

Problem:

Single @ conflicted with class declaration parser logic.

Resolution:

Introduced explicit closer:

@.close

Status:

Fixed

---

## Issue #005

### Method Close Syntax Ambiguity

Original Syntax:

/

Resolution:

Introduced explicit closer:

/.close

Status:

Fixed

# RA v1.0.1

## Major Language Improvements

Version 1.0.1 focused on usability, object manipulation, and method execution.

---

# Feature 001

## Explicit Closers

Added:

@.close
/.close

Database remained:

db.close

Loop closers:

#

!.close
?.close

Benefits:

* Cleaner parsing
* Reduced ambiguity
* Better editor support

Status:

Implemented

---

# Feature 002

## New Object Creation Syntax

Old Design:

Obj.Person == Ken

New Design:

Obj.Person.Ken

Benefits:

* Consistent dot notation
* Easier parsing
* More readable

Example:

Obj.Person.Ken

Status:

Implemented

# Feature 005

## Method Execution System

Original Design

Methods only existed as definitions.

Example:

M.Greet:
p."Hello"
/.close

No execution mechanism existed.

---

### New Syntax

Method Invocation:

Greet.run

Example:

M.Greet:
p."Hello"
/.close

Greet.run

Output:

Hello

Status:

Implemented

Implemented

# Unexpected Discovery During Testing

Test:

M.Test:
p."A"
p."B"
/.close

Test.run

Output:

A
B

Immediately after:

? For.i=1;4,

REPL returned:

3

Observation:

The parser/runtime executed a partial loop expression outside a completed block.

Likely Cause:

REPL buffering for loop blocks remains incomplete.

Status:

Known Issue

Priority:

High

Target Version:

RA v1.0.2
